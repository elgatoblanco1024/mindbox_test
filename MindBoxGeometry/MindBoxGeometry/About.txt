Использование библиотеки

1) Для создания фигуры воспользуйтесь фабрикой ShapeFactory, метод ShapeFactory.GetShape
Метод принимает массив параметров для сохранения универсальности при добавлении новых фигур
Метод обрабатывает исключения и вернет null для неверного объекта.
2) Поддерживаемые фигуры находятся в перечислении ShapeType
3) Все фигуры наследуют интерфейс I2dShape, который определяет метод вычисления площади GetSquare

Модификация библиотеки

Добавление фигуры:
1) создайте класс от I2dShape
2) добавьте новый элемент в перечисление ShapeType
3) не забудьте валидацию входных данных вашего объекта и возврат объекта из метода фабрики
4) добавьте тесткласс
